## Module Overview
- Event bus utilities wiring crawler notifications to the WebSocket gateway.

## Responsibilities
- Instantiate a shared `EventEmitter` with a generous listener limit.
- Provide a central channel (`bus.emit('event', payload)`) for run updates.

## Interfaces
- Default export: `bus` (`EventEmitter` instance).
- Emits: `'event'` with payload `{ runId, ... }`.

## Depends On
- Node.js `events` module.
- Consumers publishing events from `/internal` routes.

## Used By
- `server/routes/internal.js` to broadcast crawler events.
- `server/ws/index.js` subscribing to relay updates to WebSocket clients.
