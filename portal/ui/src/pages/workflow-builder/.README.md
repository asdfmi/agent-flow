## Module Overview
- Workflow builder feature page for composing and editing workflow definitions.

## Responsibilities
- Render the builder canvas, side panels, and step configuration forms.
- Coordinate form state via builder-specific hooks and utilities.
- Persist changes through the workflows API.

## Interfaces
- Entry: `main.jsx` bootstraps `<WorkflowBuilderPage />`.
- Components and hooks aggregated under `components/`, `hooks/`, and `utils/`.

## Depends On
- Shared components (`src/components`), workflow hooks, and API clients.
- React state management and MUI UI primitives.

## Used By
- Operators editing workflows through the portal UI.
- Other modules referencing builder utilities for validation or previews.
