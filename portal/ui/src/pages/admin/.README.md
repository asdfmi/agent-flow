## Module Overview
- Admin console page for ad-hoc CRUD access to portal data tables.

## Responsibilities
- Fetch table metadata, render generic forms, and submit mutations.
- Surface recent records and related entities for quick inspection.

## Interfaces
- Entry: `main.jsx` bootstraps `<AdminApp />`.
- Components interact with API helpers in `src/api/admin.js`.

## Depends On
- Admin API endpoints under `/api/admin`, shared components, and hooks.

## Used By
- Operators requiring direct table-level access for maintenance tasks.
