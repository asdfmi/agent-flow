## Module Overview
- Components dedicated to the workflows listing experience.

## Responsibilities
- Render workflow tables, action controls, and detail dialogs.
- Compose shared components with workflows-specific data bindings.

## Interfaces
- `WorkflowsPage` subcomponents receive props such as `workflows`, `onSelect`, `isLoading`.
- Additional helpers documented within each component file.

## Depends On
- MUI components, hooks from `src/hooks`, and API result shapes.

## Used By
- `pages/workflows/App.jsx` to assemble the workflows screen.
- Potential dashboard widgets needing workflow summaries.
