## Module Overview
- React source tree for the operator UI, covering API clients, shared components, hooks, and pages.

## Responsibilities
- Compose pages for workflows and the builder experience.
- Provide reusable UI primitives and data-fetching hooks.
- Integrate with portal REST and WebSocket endpoints.

## Interfaces
- Entry modules under `pages/*/main.jsx`.
- Hooks exported from `hooks/` (e.g., `useWorkflowRun`).
- API clients under `api/` for `/api/workflows`.

## Depends On
- React 19, MUI, Emotion, and internal API endpoints.
- Vite bundler configuration (`ui/vite.config.js`).

## Used By
- `pnpm run dev:ui` build pipeline generating `ui/dist`.
- Portal server serving the compiled assets.
