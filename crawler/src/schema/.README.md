## Module Overview
- JSON Schemas defining the accepted workflow payload format for the crawler.

## Responsibilities
- Describe workflow step structure, supported fields, and validation rules.
- Serve as the single source of truth for `WorkflowValidator`.

## Interfaces
- `workflow.schema.json` consumed via file URL by `WorkflowValidator`.
- Schema drafts follow JSON Schema conventions (AJV compatible).

## Depends On
- Maintained alongside crawler engine updates to keep validation aligned.
- AJV loader in `services/workflow-validator.js`.

## Used By
- `WorkflowValidator` when checking incoming workflows.
- Documentation or tooling that needs the formal workflow contract.
